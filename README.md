# 🚗 MyChoize – Premium Car Rental (Frontend Project)

## 📌 Project Overview
MyChoize – Premium Car Rental is a frontend web application designed to simulate an online car rental platform. The website provides a modern user interface where users can explore different cars and experience a premium car rental service layout.

This project focuses on creating a responsive and visually appealing design using core web technologies.

## 📄 Pages Included 
- Home Page
- About Page
- Contact Page
- Services Page
- Booking Confirmation Page
- Model Confirmation Page
- Login page
- Sign up Page
- Profile Page

## 🛠️ Technologies Used
- HTML5 – Structure of the website
- CSS3 – Styling and layout
- JavaScript – Interactive features
- Font Awesome – Icons

## ✨ Key Features
- Responsive car rental website design
- Clean and modern user interface
- Car listing and showcase sections
- Navigation bar and hero section
- Service and footer sections
- User-friendly layout
